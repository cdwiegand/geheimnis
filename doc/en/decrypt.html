Geheimnis Documentation

  1. Choose your profile

    Select your profile to use for decrypting and/or verifying.

  2. Enter the file to act on
  3. Select options

    If you are verifying a detached signature, check Detached Signature, and enter the filename or choose the file to use as a signature. If you are decrypting, enter the filename or choose the file to save the decrypted data to. If you are doing neither, leave the options alone.

  4. Click Go.
